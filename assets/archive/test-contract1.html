The Problem

Multi-agent systems today share context, files, and credentials. One rogue agent contaminates everything. This is the supply chain attack of AI workflows.
  • Shared file access = lateral movement risk
  • No contract boundaries = scope creep
  • No audit trail = no forensics
  • Current tools assume trust between agents
  • We need isolation by default
Innovation pillar — frame agent isolation as security architecture

The Architecture

git worktree add ../avp-css-effects dev/human-a/css-effects git worktree add ../avp-narration dev/human-b/narration # Each agent: isolated filesystem, exclusive file ownership # Interface contracts define the only shared surface

  • Contracts = zero-trust API boundaries
  • Git history = audit trail
