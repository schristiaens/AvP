# AvP Repository Instructions

## Agent Workflow

**Before doing any work in this repository, read `AGENT-WORKFLOW.md` at the repo root.** It defines the coordination strategy for multi-developer agent work:

- You are likely in a worktree. Stay within your work item's file scope.
- Don't touch shared types/interfaces without explicit developer approval.
- Rebase from main daily. Merge immediately on completion.
- Respect the contracts-before-work protocol described in the plan.

All agents working in this repo must follow that plan.
